## Description
Base python template for projects

## Setup
Init virtualenv, install requirements, install pre-commit hooks
> make setup


## Useful commands

Run app
> make run

Run Tests
> make test

Run tests with coverage
> make coverage

Clean cache
> make clean

Mypy
> make mypy

Flake8
> make lint

Isort && Black
> make black
